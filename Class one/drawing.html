Drawing

Going back to what I have tried before, I realize that drawing is almost the only thing that I persist in for so many years. I used to learn piano, tennis, dance, calligraphy, and so on, but I gave up all these interests halfway through. ( A random fun fact ) My parents used to not like me spending too much time drawing. I remembered when I was a kid, I asked them if I can go to a drawing time several times and they never agreed. I was a rebellious kid. the more I was asked not to do something, the more I want to try doing it, so I found free time and learned how to draw by myself secretly .
